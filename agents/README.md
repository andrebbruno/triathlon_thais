# Agents - Atalhos

Use os prefixos abaixo para iniciar a conversa com cada agente.

| Agente | Prefixo | Atalho |
| --- | --- | --- |
| Head (geral) | @head | agents/head/START.md |
| Coach Tri (treino) | @coachtri | agents/coachTri/START.md |
| IT Tech (scripts) | @it_tech | agents/IT_tech/START.md |
| Nutricao | @nutricao | agents/nutricao/START.md |
| Fisio | @fisio | agents/fisio/START.md |

Como usar:
1. Abra o START.md do agente desejado.
2. Copie o prefixo e cole no inicio da sua mensagem.
3. O agente vai ler os arquivos de memoria indicados no START.

Memorias:
- memory/CORE.md: contexto compartilhado
- memory/*: memorias especificas por agente
- sessions/*: resumo curto da ultima sessao por agente
